üíª How to Install Composer

Windows

  • Download Composer from the official Composer website.
  • Run the Composer-Setup.exe installer.
  • During setup, locate your PHP executable (e.g., C:/xampp/php/php.exe).
  • After installation, open Command Prompt and type: composer

macOS / Linux

Open Terminal and run:

php -r "copy('https://getcomposer.org/installer', 'composer-setup.php');"

php composer-setup.php

php -r "unlink('composer-setup.php');"

To install globally:

sudo mv composer.phar /usr/local/bin/composer

Verify installation:

composer

üöÄ Basic Composer Usage

Step 1: Create composer.json

{
"require": {
"monolog/monolog": "2.0.*"
}
}

Step 2: Install Dependencies

Run this in your project folder:

composer install

Step 3: Update Dependencies

composer update

Step 4: Autoload Packages

Include this in your PHP file:

require_once 'vendor/autoload.php';

üß™ Sample Project: Logging with Monolog

1. Initialize Project

composer init

2. Add Monolog

composer require monolog/monolog

3. Create index.php

<?php
require_once 'vendor/autoload.php';

use Monolog\Logger;
use Monolog\Handler\StreamHandler;

$log = new Logger('student_log');
$log->pushHandler(new StreamHandler('app.log', Logger::WARNING));

$log->warning('This is a warning');
$log->error('This is an error');
?>

4. Run the File

php index.php

5. Check app.log for Output

Open the app.log file in your project folder to see the logged messages.
