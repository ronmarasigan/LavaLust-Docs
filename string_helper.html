String Helpers

The String Helper provides functions for common string manipulation tasks in your LavaLust PHP Framework application.

str_insert()

Description

Replaces placeholders in a string with corresponding values from an associative array.

Syntax

str_insert(array $keyValue, string $string)

Parameters

Parameter Type Description Default
$keyValue array Associative array of search => replace pairs Required
$string string The string containing placeholders to replace Required

Return Value

string - The string with replacements made

Example Usage

$data = ['{name}' => 'John', '{age}' => '30'];
$template = "Hello {name}, you are {age} years old.";
echo str_insert($data, $template);
// Output: Hello John, you are 30 years old.

str_between()

Description

Extracts all substrings between left and right delimiters.

Syntax

str_between(string $left, string $right, string $string)

Parameters

Parameter Type Description Default
$left string Left delimiter Required
$right string Right delimiter Required
$string string String to search in Required

Return Value

array - Array of matched strings between delimiters

Example Usage

$text = "Sample [text] with [multiple] matches";
$results = str_between('[', ']', $text);
// Returns: ['text', 'multiple']

str_after() / str_before()

Description

Extracts substring after or before the first occurrence of a search string.

Syntax

str_after(string $search, string $string)
str_before(string $search, string $string)

Parameters

Parameter Type Description Default
$search string String to search for Required
$string string String to search in Required

Return Value

string - The substring after/before the search string

Example Usage

$email = "user@example.com";
$username = str_before('@', $email); // Returns "user"
$domain = str_after('@', $email);    // Returns "example.com"

str_limitwords() / str_limitchars()

Description

Truncates a string to a specified number of words or characters.

Syntax

str_limitwords(string $string, int $limit = 10, string $end = '...')
str_limitchars(string $string, int $limit = 100, string $end = '...')

Parameters

Parameter Type Description Default
$string string String to truncate Required
$limit int Maximum words/chars to keep 10 (words) / 100 (chars)
$end string String to append if truncated '...'

Return Value

string - The truncated string

Example Usage

$text = "This is a sample text string for demonstration purposes.";

// Limit to 5 words
echo str_limitwords($text, 5);
// Output: "This is a sample text..."

// Limit to 20 characters
echo str_limitchars($text, 20);
// Output: "This is a sample..."

strip_slashes() / strip_quotes() / quotes_to_entities()

Description

Various string sanitization functions.

Syntax

strip_slashes(mixed $str)
strip_quotes(string $str)
quotes_to_entities(string $str)

Parameters

Function Description
strip_slashes Removes slashes from a string or array (recursive)
strip_quotes Removes single and double quotes from a string
quotes_to_entities Converts quotes to HTML entities

Example Usage

$text = "This \"string\" has 'quotes' and \\slashes\\";

echo strip_slashes($text);      // "This "string" has 'quotes' and slashes"
echo strip_quotes($text);       // "This string has quotes and \slashes\"
echo quotes_to_entities($text); // "This "string" has 'quotes' and \slashes\"

reduce_double_slashes() / reduce_multiples()

Description

Cleans up strings with repeated characters.

Syntax

reduce_double_slashes(string $str)
reduce_multiples(string $str, string $character = ',', bool $trim = FALSE)

Parameters

Function Description
reduce_double_slashes Reduces multiple slashes to single slashes (except in http://)
reduce_multiples Reduces multiple instances of a specified character

Example Usage

$url = "http://example.com//path//to//file";
echo reduce_double_slashes($url); // "http://example.com/path/to/file"

$list = "a,,b,c,,,d";
echo reduce_multiples($list);     // "a,b,c,d"

random_string()

Description

Generates random strings of various types.

Syntax

random_string(string $type = 'alnum', int $len = 8)

Types

Type Description
basic Random number
alpha Alphabetic characters only
alnum Alphanumeric characters (default)
numeric Numeric characters only
nozero Numeric characters without zero
md5 MD5 hash
sha1 SHA1 hash

Example Usage

echo random_string('alpha', 10);  // "XcFhJiKlmN"
echo random_string('numeric', 6); // "384729"
echo random_string('md5');        // "d41d8cd98f00b204e9800998ecf8427e"

increment_string()

Description

Increments a string by appending a number or incrementing an existing number.

Syntax

increment_string(string $str, string $separator = '_', int $first = 1)

Example Usage

echo increment_string('file');       // "file_1"
echo increment_string('file_1');    // "file_2"
echo increment_string('file', '-'); // "file-1"

alternator()

Description

Alternates between values with each call.

Syntax

alternator(mixed ...$args)

Example Usage

$alt = alternator('one', 'two', 'three');
echo $alt(); // "one"
echo $alt(); // "two"
echo $alt(); // "three"
echo $alt(); // "one" (cycles back to start)

repeater()

Description

Repeats a string a specified number of times.

Syntax

repeater(string $data, int $num = 1)

Example Usage

echo repeater('abc', 3); // "abcabcabc"
echo repeater('-', 10); // "----------"
