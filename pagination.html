Overview

The Pagination class helps generate page links for dividing records into multiple pages. It supports different themes such as Default, Bootstrap, and Tailwind.

Properties

$total_rows int

Total number of records.

$per_page int

Number of records shown per page.

$current_page int

Currently active page.

$theme string

Theme used for pagination links (default, bootstrap, tailwind).

Methods

Initializes pagination with optional configuration.

Sets the pagination theme.

Returns: string

Usage Examples

Default Pagination

$config = [
  'total_rows' => 100,
  'per_page' => 10,
  'current_page' => 1
];

$pager = new Pagination($config);
echo $pager->paginate();

Bootstrap Theme

$pager = new Pagination([
  'total_rows' => 50,
  'per_page' => 5,
  'current_page' => 2,
  'theme' => 'bootstrap'
]);
echo $pager->paginate();

Tailwind Theme

$pager = new Pagination();
$pager->initialize([
  'total_rows' => 75,
  'per_page' => 15,
  'current_page' => 3
]);
$pager->set_theme('tailwind');
echo $pager->paginate();

Best Practices

Ensure that $total_rows always matches the actual number of rows from your database query.

Keep $per_page consistent between your query and pagination setup to avoid mismatched results.

Always call initialize() or pass configuration in the constructor before calling paginate(), otherwise pagination may not render correctly.
