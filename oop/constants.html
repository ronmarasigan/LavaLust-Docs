üîç What Are Constants?

A constant is a name or identifier for a fixed value that cannot change during the execution of a script. Unlike variables, constants:

  • Do not start with $
  • Are automatically global
  • Are immutable once defined

üõ† Defining Constants

Using define()

define("SITE_NAME", "CodeAcademy");
echo SITE_NAME; // Output: CodeAcademy

Using const

const VERSION = "1.0.0";
echo VERSION; // Output: 1.0.0

üì¶ Constant Arrays (PHP 7+)

define("FRUITS", ["Apple", "Banana", "Cherry"]);
echo FRUITS[1]; // Output: Banana

Note: You can only use define() for array constants. const supports only scalar values.

üåê Scope of Constants

Constants are global by default and accessible anywhere in the script:

define("GREETING", "Hello!");

function sayHello() {
    echo GREETING;
}

sayHello(); // Output: Hello!

üèõ Class Constants

Use const inside classes to define constants:

class Config {
    const MAX_USERS = 100;
}

echo Config::MAX_USERS; // Output: 100

üìã Best Practices

  • Use UPPERCASE for constant names
  • Group related constants in classes
  • Use constants for config values, limits, paths, etc.
