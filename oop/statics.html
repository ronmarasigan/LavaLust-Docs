üìò What Are Static Methods and Properties?

Static methods and properties belong to the class itself, not to any object instance. They can be accessed without creating an object.

üõ† Syntax

class Math {
    public static $pi = 3.14159;

    public static function square($n) {
        return $n * $n;
    }
}

üì¶ Accessing Static Members

echo Math::$pi;               // Output: 3.14159
echo Math::square(5);         // Output: 25

üîÅ Accessing Inside the Class

class Counter {
    public static $count = 0;

    public static function increment() {
        self::$count++;
    }

    public static function getCount() {
        return self::$count;
    }
}

Counter::increment();
echo Counter::getCount();     // Output: 1

üß¨ Inheritance and Static Members

class Base {
    public static $message = "Hello from Base";
}

class Derived extends Base {}

echo Derived::$message;       // Output: Hello from Base

üìã Key Points

  • Use static keyword to declare static members.
  • Use self:: to access static members inside the class.
  • Use ClassName:: to access them outside.
  • $this is not available in static methods.
  • Static members are shared across all instances.
