üîç What Are Classes and Objects?

In PHP, a class is a blueprint for creating objects. It defines properties (variables) and methods (functions) that describe the behavior and state of the object.

An object is an instance of a class. It holds actual data and can perform actions defined by the class.

üß± Defining a Class

Use the class keyword followed by the class name and curly braces:

class Car {
    public $brand;
    public $model;

    public function setBrand($brand) {
        $this->brand = $brand;
    }

    public function getBrand() {
        return $this->brand;
    }
}

üöó Creating Objects

Use the new keyword to create an object from a class:

$myCar = new Car();
$myCar->setBrand("Toyota");
echo $myCar->getBrand();

‚öôÔ∏è Constructor Method

The __construct() method automatically runs when an object is created:

class Car {
    public $brand;
    public $model;

    public function __construct($brand, $model) {
        $this->brand = $brand;
        $this->model = $model;
    }

    public function displayInfo() {
        echo "Car: $this->brand $this->model";
    }
}

$car = new Car("Toyota", "Corolla");
$car->displayInfo();

üîë The $this Keyword

$this refers to the current object and is used to access its properties and methods:

public function setModel($model) {
    $this->model = $model;
}

üìö Summary

  • Class: Blueprint for objects.
  • Object: Instance of a class.
  • Constructor: Initializes object properties.
  • $this: Refers to the current object.
