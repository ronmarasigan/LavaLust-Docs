üîç Conditional Statements

Conditional statements control the flow of execution based on logical conditions. They allow your program to respond dynamically to different inputs or states.

if / else

Definition: Executes a block of code if a condition is true. You can chain multiple conditions using else if and provide a fallback with else.

Use Case: Grading systems, login validation, feature toggles.

int score = 85;

if (score >= 90) {
    Console.WriteLine("Excellent!");
} else if (score >= 75) {
    Console.WriteLine("Good job!");
} else {
    Console.WriteLine("Keep practicing.");
}

switch

Definition: Selects one of many code blocks to execute based on the value of a variable. Ideal for discrete, known values.

Use Case: Menu selection, command parsing, grade evaluation.

char grade = 'B';

switch (grade) {
    case 'A':
        Console.WriteLine("Outstanding!");
        break;
    case 'B':
        Console.WriteLine("Very good!");
        break;
    case 'C':
        Console.WriteLine("Fair.");
        break;
    default:
        Console.WriteLine("Invalid grade.");
        break;
}

üîÅ Looping Constructs

Loops allow you to repeat a block of code multiple times. They‚Äôre essential for tasks like iteration, searching, and repeated calculations.

for Loop

Definition: A counted loop that runs a fixed number of times. Includes initialization, condition, and increment.

Use Case: Iterating over arrays, generating patterns, fixed repetitions.

for (int i = 1; i <= 5; i++) {
    Console.WriteLine("Iteration " + i);
}

while Loop

Definition: Runs as long as the condition is true. The condition is checked before each iteration.

Use Case: Waiting for user input, polling, conditional repetition.

int count = 0;

while (count < 3) {
    Console.WriteLine("Count is " + count);
    count++;
}

do-while Loop

Definition: Similar to while, but guarantees at least one execution before checking the condition.

Use Case: Menu loops, retry prompts, post-condition checks.

int number = 1;

do {
    Console.WriteLine("Number: " + number);
    number++;
} while (number <= 3);

foreach Loop

Definition: Iterates over each element in a collection (like arrays or lists). Simplifies access without index management.

Use Case: Displaying items, filtering data, read-only iteration.

string[] fruits = { "Apple", "Banana", "Cherry" };

foreach (string fruit in fruits) {
    Console.WriteLine(fruit);
}

üìò Summary

  • Conditionals guide decision-making: use if/else for flexible logic and switch for structured value-based branching.
  • Loops enable repetition: use for for counted iterations, while for pre-checked conditions, do-while for guaranteed execution, and foreach for clean collection traversal.

Mastering control flow is essential for building responsive, efficient programs. Encourage students to experiment with nested loops and compound conditions to deepen their understanding.

Educator Tip: Assign mini-projects like a number guessing game, dynamic menu system, or array analyzer to reinforce both conditionals and loops in practical scenarios.
