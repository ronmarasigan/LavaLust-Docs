ðŸ§¾ Basic Syntax

C# programs are structured into namespaces, classes, and methods. Here's a simple example:

using System;

class Program {
    static void Main() {
        Console.WriteLine("Hello, World!");
    }
}
  • using System; imports the System namespace.
  • Main() is the entry point of the program.
  • Console.WriteLine() prints output to the console.

ðŸ“¦ Data Types

C# is statically typed. Common data types include:

Type Description Example
int Integer numbers int age = 30;
double Floating-point numbers double price = 19.99;
char Single character char grade = 'A';
bool Boolean value bool isActive = true;
string Text data string name = "Ron";

ðŸ”§ Operators

C# supports various operators for computation and logic:

Category Operators Example
Arithmetic +, -, *, /, % int sum = a + b;
Comparison ==, !=, >, <, >=, <= if (a > b)
Logical &&, ||, ! if (a > 0 && b < 10)
Assignment =, +=, -=, *=, /= x += 5;
Note: Operator precedence matters! For example, multiplication and division are evaluated before addition and subtraction unless parentheses are used.

ðŸ“˜ Summary

  • C# syntax is clean, structured, and similar to other C-style languages.
  • It uses a rich set of data types to represent different kinds of values.
  • Operators allow you to perform arithmetic, comparisons, logical decisions, and assignments.

Mastering these basics is essential before moving on to control structures, methods, and object-oriented programming. Encourage students to write small programs that use variables, operators, and console output to reinforce these concepts.
