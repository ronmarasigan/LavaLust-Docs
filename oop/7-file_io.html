üìÇ Overview

File I/O (Input/Output) in C# is handled via the System.IO namespace. It allows reading from and writing to files on disk.

Common classes include:

  • File ‚Äì static methods for file operations
  • FileStream ‚Äì low-level stream access
  • StreamReader / StreamWriter ‚Äì text-based file access

‚öôÔ∏è File Modes

File modes define how a file is opened or created. Here's a comparison:

Mode Description C# Equivalent
r Read-only. File must exist. FileMode.Open
w Write-only. Creates new or overwrites. FileMode.Create
w+ Read and write. Overwrites if exists. FileMode.Create + FileAccess.ReadWrite
a Append. Writes to end of file. FileMode.Append
a+ Read and append. FileMode.OpenOrCreate + FileAccess.ReadWrite

üìù Create

Create and write to a file using StreamWriter:

string path = "data.txt";
using (StreamWriter writer = new StreamWriter(path)) {
  writer.WriteLine("New file created.");
  writer.WriteLine("Welcome to File I/O.");
}

üìñ Read

Read file contents using StreamReader:

string path = "data.txt";
using (StreamReader reader = new StreamReader(path)) {
  string line;
  while ((line = reader.ReadLine()) != null) {
    Console.WriteLine(line);
  }
}

‚úèÔ∏è Update

Update a file by reading, modifying, and rewriting:

string path = "data.txt";

// Read all lines
string[] lines = File.ReadAllLines(path);

// Modify content (example: replace a word)
for (int i = 0; i < lines.Length; i++) {
  lines[i] = lines[i].Replace("Welcome", "Hello");
}

// Write back to file
File.WriteAllLines(path, lines);
üí° Tip: For large files, use StreamReader and StreamWriter with a temporary file to avoid memory overhead.

üóëÔ∏è Delete

Delete a file using File.Delete():

string path = "data.txt";

if (File.Exists(path)) {
  File.Delete(path);
  Console.WriteLine("File deleted.");
} else {
  Console.WriteLine("File not found.");
}
‚ö†Ô∏è Warning: File.Delete() is irreversible. Always confirm before deleting user data.

  • CRUD operations modularized:
