üíª How to Install Git

Windows

  • Download from Git for Windows.
  • Run the installer and follow the setup wizard.
  • Open Git Bash or Command Prompt and type: git --version

macOS

  • Open Terminal and type: git --version
  • If not installed, follow the prompt to install Xcode Command Line Tools.
  • Or use Homebrew: brew install git

Linux

  • Debian/Ubuntu: sudo apt update then sudo apt install git
  • Fedora: sudo dnf install git
  • Arch Linux: sudo pacman -S git

üöÄ Basic Git Commands & Functions

Task Command Description
Configure Git git config --global user.name "Your Name"
git config --global user.email "you@example.com"		 Set your name and email
Initialize repo git init Start a new Git repo
Clone repo git clone <repo-url> Copy remote repo
Check status git status See changes
Stage changes git add <file> or git add . Prepare files for commit
Commit changes git commit -m "Your message" Save changes
View history git log or git log --oneline Show commits
Create branch git branch <branch-name> Make a new branch
Switch branch git checkout <branch-name> Move to branch
Merge branch git merge <branch-name> Merge changes
Push to remote git push origin <branch-name> Upload changes
Pull from remote git pull origin <branch-name> Download updates
Delete branch git branch -d <branch-name> Remove branch
