üìò What Are Superglobals?

Superglobals are built-in variables in PHP that are always accessible, regardless of scope. You can use them inside functions, classes, or files without needing to declare them global.

üìã Common Superglobals

1. $GLOBALS

Contains all global variables in an associative array.

$x = 5;
$y = 10;

function multiply() {
    $GLOBALS['z'] = $GLOBALS['x'] * $GLOBALS['y'];
}

multiply();
echo $z; // Outputs: 50

2. $_SERVER

Provides information about headers, paths, and script locations.

echo $_SERVER['SERVER_NAME'];
echo $_SERVER['REQUEST_METHOD'];

3. $_GET

Collects data sent via URL query parameters.

// URL: index.php?name=Ron
echo $_GET['name']; // Outputs: Ron

4. $_POST

Collects form data sent via POST method.

// HTML Form
<form method="post">
  <input type="text" name="username">
</form>

// PHP
echo $_POST['username'];

5. $_REQUEST

Contains data from $_GET, $_POST, and $_COOKIE.

echo $_REQUEST['username'];

6. $_SESSION

Stores session variables.

session_start();
$_SESSION['user'] = "Ron";
echo $_SESSION['user'];

7. $_COOKIE

Stores cookie values.

echo $_COOKIE['theme'];

8. $_FILES

Handles file uploads.

$_FILES['upload']['name'];
$_FILES['upload']['tmp_name'];

9. $_ENV

Contains environment variables.

echo $_ENV['PATH'];
