⚙️ PHP Operators

📘 What Are Operators?

Operators are symbols that perform operations on variables and values. PHP supports many types of operators, each designed for specific tasks like math, comparison, logic, and assignment.

➕ Arithmetic Operators

$x = 10;
$y = 3;

echo $x + $y; // Addition → 13
echo $x - $y; // Subtraction → 7
echo $x * $y; // Multiplication → 30
echo $x / $y; // Division → 3.33
echo $x % $y; // Modulus → 1
echo $x ** $y; // Exponentiation → 1000

📝 Assignment Operators

$x = 5;
$x += 3; // $x = $x + 3 → 8
$x -= 2; // $x = $x - 2 → 6
$x *= 4; // $x = $x * 4 → 24
$x /= 6; // $x = $x / 6 → 4
$x %= 3; // $x = $x % 3 → 1

🔍 Comparison Operators

$a = 5;
$b = "5";

var_dump($a == $b);  // true (equal)
var_dump($a === $b); // false (not identical)
var_dump($a != $b);  // false
var_dump($a !== $b); // true
var_dump($a < 10);   // true
var_dump($a >= 5);   // true

🔗 Logical Operators

$x = true;
$y = false;

var_dump($x && $y); // false
var_dump($x || $y); // true
var_dump(!$x);      // false

🧵 String Operators

$first = "Hello";
$second = "World";

echo $first . " " . $second; // Concatenation → Hello World
$first .= " PHP";
echo $first; // → Hello PHP

❓ Ternary Operator

$age = 18;
$status = ($age >= 18) ? "Adult" : "Minor";
echo $status; // → Adult

🧪 Practice Activity

Create a PHP file named operators.php. Use each type of operator to manipulate variables and display the results. Try combining logical and comparison operators in an if statement.
