üìò What Is a Variable?

A variable is a container for storing data. In PHP, variables start with a dollar sign ($) followed by the variable name.

$name = "Ron";
$age = 30;

üß† Naming Rules

  • Must start with $ followed by a letter or underscore
  • Can contain letters, numbers, and underscores
  • Cannot start with a number
  • Are case-sensitive ($name ‚â† $Name)

üåç Variable Scope

  • Global: Declared outside functions
  • Local: Declared inside functions
  • Static: Retains value across function calls

üì¶ PHP Data Types

1. String

A sequence of characters enclosed in quotes.

$greeting = "Hello, World!";
$name = 'Ron';

2. Integer

Whole numbers, positive or negative.

$age = 30;
$year = -2025;

3. Float (Double)

Numbers with decimal points.

$price = 19.99;
$temperature = -5.5;

4. Boolean

Represents true or false.

$isStudent = true;
$isAdmin = false;

5. Array

Stores multiple values in one variable.

$colors = array("red", "green", "blue");
$person = array("name" => "Ron", "age" => 30);

6. Object

Instance of a class containing properties and methods.

class Car {
  public $model;
  public $color;
}

$myCar = new Car();
$myCar->model = "Toyota";
$myCar->color = "Red";

7. NULL

Represents a variable with no value.

$data = NULL;

üîç Checking Data Type

Use var_dump() to inspect type and value:

$x = "Hello";
var_dump($x);

Create a PHP file named types.php. Declare one variable for each data type and use var_dump() to display its type and value.
