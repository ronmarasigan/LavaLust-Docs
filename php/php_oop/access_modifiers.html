üîç What Are Access Modifiers?

Access modifiers in PHP control the visibility of class properties and methods. They determine where members can be accessed ‚Äî inside the class, by subclasses, or externally.

üü¢ Public

Public members can be accessed from anywhere ‚Äî inside the class, by subclasses, and from outside.

class Fruit {
    public $name;

    public function setName($n) {
        $this->name = $n;
    }

    public function getName() {
        return $this->name;
    }
}

$mango = new Fruit();
$mango->setName("Mango");
echo $mango->getName();

üî¥ Private

Private members can only be accessed within the class itself. Not accessible from outside or by subclasses.

class Secret {
    private $code = "1234";

    private function getCode() {
        return $this->code;
    }

    public function reveal() {
        return $this->getCode(); // Allowed internally
    }
}

$obj = new Secret();
echo $obj->reveal();      // ‚úÖ Works
echo $obj->code;          // ‚ùå Error
echo $obj->getCode();     // ‚ùå Error

üü° Protected

Protected members are accessible within the class and by subclasses, but not from outside.

class ParentClass {
    protected $message = "Hello from Parent";

    protected function showMessage() {
        return $this->message;
    }
}

class ChildClass extends ParentClass {
    public function getMessage() {
        return $this->showMessage(); // Allowed in subclass
    }
}

$obj = new ChildClass();
echo $obj->getMessage();     // ‚úÖ Works
echo $obj->message;          // ‚ùå Error

üìå Summary Table

  • Public: Accessible from anywhere.
  • Private: Accessible only within the class.
  • Protected: Accessible within the class and subclasses.
