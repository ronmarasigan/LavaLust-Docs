üìò What Is Form Handling?

Form handling is the process of collecting and processing user input submitted via HTML forms. PHP uses superglobals like $_GET and $_POST to retrieve form data.

üìù Creating an HTML Form

<form method="post" action="process.php">
  <label>Name:</label>
  <input type="text" name="name" required><br>
  <label>Email:</label>
  <input type="email" name="email" required><br>
  <input type="submit" value="Submit">
</form>

üì• Handling Form Data in PHP

// process.php
$name = $_POST['name'];
$email = $_POST['email'];

echo "Welcome, " . htmlspecialchars($name);
echo "Your email is: " . htmlspecialchars($email);

Note: Use htmlspecialchars() to prevent XSS attacks.

üîç Using GET Method

<form method="get" action="process_get.php">
  <input type="text" name="query">
  <input type="submit">
</form>
// process_get.php
echo $_GET['query'];

‚úÖ Basic Validation

if (empty($_POST['name'])) {
    echo "Name is required.";
} elseif (!filter_var($_POST['email'], FILTER_VALIDATE_EMAIL)) {
    echo "Invalid email format.";
}
