👋 Introduction to PHP

📘 What Is PHP?

PHP stands for Hypertext Preprocessor. It is a widely-used, open-source server-side scripting language designed for web development. PHP can be embedded directly into HTML and is executed on the server to generate dynamic content.

⚙️ How PHP Works

  1. The user requests a page via a browser.
  2. The web server (e.g., Apache or Nginx) receives the request.
  3. The server passes the request to the PHP interpreter.
  4. PHP executes the script, optionally interacts with a database, and returns HTML to the browser.

👨‍💻 Your First PHP Script

Let’s write a simple “Hello, World!” program in PHP:

<?php
echo "Hello, World!";
?>

This script uses echo to output text. The <?php and ?> tags tell the server to interpret the enclosed code as PHP.

📦 What Can PHP Do?

  • Generate dynamic page content
  • Collect form data
  • Send and receive cookies
  • Interact with databases (e.g., MySQL)
  • Manage sessions
  • Build entire web applications (CMS, e-commerce, etc.)
