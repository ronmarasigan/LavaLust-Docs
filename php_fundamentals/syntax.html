üìò Basic Syntax

PHP code is written between <?php and ?> tags. These tags tell the server to interpret the enclosed code as PHP.

<?php
echo "Hello, PHP!";
?>

üìÑ File Structure

A typical PHP file contains both HTML and PHP code. The file extension must be .php.

<!DOCTYPE html>
<html>
<body>

<h1>My First PHP Page</h1>
<?php
echo "This is dynamic content!";
?>

</body>
</html>

üìå Syntax Rules

  • PHP statements end with a semicolon ;
  • Variables start with $ and are case-sensitive
  • Keywords like if, echo, while are not case-sensitive
  • PHP can be embedded directly into HTML

   comment block */
