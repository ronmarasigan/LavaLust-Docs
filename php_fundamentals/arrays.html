An array is a special variable that can hold multiple values under a single name. You can access values using numeric or named keys.

Indexed arrays use numeric keys starting from 0.

Associative arrays use named keys to store values.

  "city" => "Calapan"
];
echo $person["name"]; // Outputs: Ron

Multidimensional arrays contain other arrays as elements.

];
