üìò What Is Control Flow?

Control flow determines the order in which statements are executed in a program. PHP provides conditional statements and loops to control the flow based on logic and conditions.

‚úÖ Conditional Statements

1. if Statement

$x = 10;
if ($x > 0) {
    echo "Positive number";
}

2. if...else Statement

$x = -5;
if ($x > 0) {
    echo "Positive";
} else {
    echo "Negative";
}

3. if...elseif...else Statement

$day = "Monday";
if ($day == "Saturday") {
    echo "Weekend";
} elseif ($day == "Sunday") {
    echo "Weekend";
} else {
    echo "Weekday";
}

4. switch Statement

$grade = "B";
switch ($grade) {
    case "A":
        echo "Excellent";
        break;
    case "B":
        echo "Good";
        break;
    case "C":
        echo "Fair";
        break;
    default:
        echo "Invalid grade";
}

üîÅ Looping Statements

1. while Loop

$i = 1;
while ($i <= 5) {
    echo $i;
    $i++;
}

2. do...while Loop

$i = 1;
do {
    echo $i;
    $i++;
} while ($i <= 5);

3. for Loop

for ($i = 1; $i <= 5; $i++) {
    echo $i;
}

4. foreach Loop

$colors = ["red", "green", "blue"];
foreach ($colors as $color) {
    echo $color;
}
