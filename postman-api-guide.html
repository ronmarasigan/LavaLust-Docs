🧪 Using Postman with PHP API Controller

This guide shows how to test each endpoint of your PHP API using Postman, either in VS Code or the standalone app.

🔐 1. Login

POST /Api_controller/login

Request Body

{
  "username": "admin",
  "password": "your_password"
}

Expected Response

{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6...",
  "refresh_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6..."
}

🔓 2. Logout

POST /Api_controller/logout

Request Body

{
  "refresh_token": "your_refresh_token"
}

Expected Response

{
  "message": "Logged out"
}

👥 3. List Users

GET /Api_controller/list

Headers

Authorization: Bearer your_access_token

Expected Response

[
  {
    "id": 1,
    "username": "admin",
    "email": "admin@example.com",
    "role": "admin",
    "created_at": "2025-08-01 12:00:00"
  },
  ...
]

🧑‍💻 4. Create User

POST /Api_controller/create

Headers

Authorization: Bearer your_access_token
Content-Type: application/json

Request Body

{
  "username": "newuser",
  "email": "newuser@example.com",
  "password": "secure123",
  "role": "user"
}

✏️ 5. Update User

PUT /Api_controller/update/{id}

Headers

Authorization: Bearer your_access_token
Content-Type: application/json

Request Body

{
  "username": "updateduser",
  "email": "updated@example.com",
  "role": "editor"
}

🗑️ 6. Delete User

DELETE /Api_controller/delete/{id}

Headers

Authorization: Bearer your_access_token

🙋 7. Profile

GET /Api_controller/profile

Headers

Authorization: Bearer your_access_token

🔄 8. Refresh Token

POST /Api_controller/refresh

Request Body

{
  "refresh_token": "your_refresh_token"
}
💡 Tip: Use Postman collections and environment variables to streamline testing and token management.
