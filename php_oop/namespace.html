ðŸ“˜ What Is a Namespace?

A namespace is a way to group related classes, functions, and constants to avoid name collisions and improve code organization. Introduced in PHP 5.3, namespaces are similar to directories in a file system.

ðŸ›  Declaring a Namespace

<?php
namespace Html;

class Table {
    public $title = "";
    public $numRows = 0;

    public function message() {
        echo "<p>Table '{$this->title}' has {$this->numRows} rows.</p>";
    }
}
?>

<?php
require 'Html/Table.php';

$table->title = "My Table";
$table->numRows = 5;
$table->message();
?>

ðŸ“‹ Key Points

  • Declare namespaces at the top of the file.
  • Use fully qualified names or use keyword to access namespaced elements.
  • Namespaces can contain classes, interfaces, functions, and constants.
  • Use aliases to simplify long namespace paths.
