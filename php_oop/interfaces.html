ðŸ“˜ What Is an Interface?

An interface defines a contract that classes must follow. It specifies method signatures without implementing them. Interfaces promote flexibility and polymorphism in object-oriented design.

ðŸ›  Syntax

interface Logger {
    public function log(string $message);
}

ðŸ§± Implementing an Interface

class FileLogger implements Logger {
    public function log(string $message) {
        echo "Logging to file: $message";
    }
}

$logger = new FileLogger();
$logger->log("System started.");

ðŸ“‹ Rules

  • All methods in an interface must be public.
  • Interfaces cannot contain method bodies.
  • Classes must implement all methods declared in the interface.
  • Interfaces can contain constants.
  • Interfaces can be extended by other interfaces.

ðŸ”€ Multiple Interfaces

interface Printable {
    public function print();
}

interface Scannable {
    public function scan();
}

class MultiFunctionPrinter implements Printable, Scannable {
    public function print() {
        echo "Printing document...";
    }

    public function scan() {
        echo "Scanning document...";
    }
}
